## 一、概述

### 项目背景

随着互联网技术的快速发展，网络安全问题日益突出。传统的明文传输方式存在数据泄露、篡改、伪造等严重安全风险。SSL/TLS协议作为目前应用最广泛的网络安全协议，通过加密通信、身份认证和数据完整性校验等机制，为网络通信提供了可靠的安全保障。

OpenSSL是一个功能强大的开源密码库，实现了SSL/TLS协议的完整功能，支持多种加密算法和证书管理。本项目基于OpenSSL库，设计并实现了一套完整的安全通信系统，包括原生TCP通信、SSL单向认证和SSL双向认证三个层次，深入理解和实践SSL/TLS协议的核心原理。

### 项目意义

本项目的意义在于：
1. 深入理解SSL/TLS协议的架构和工作原理
2. 掌握OpenSSL库的使用方法和安全编程实践
3. 实现从明文通信到加密通信的完整过渡
4. 学会使用Wireshark分析网络流量，验证加密效果
5. 培养团队协作开发和安全意识

#### 3.2 模块详细设计

##### 3.2.1 证书管理模块

**证书生成流程：**
1. 生成CA根证书私钥：`openssl genrsa -out ca.key 2048`
2. 生成CA根证书：`openssl req -new -x509 -days 365 -key ca.key -out ca.crt`
3. 生成服务器私钥：`openssl genrsa -out server.key 2048`
4. 生成服务器证书请求：`openssl req -new -key server.key -out server.csr`
5. CA签发服务器证书：`openssl x509 -req -days 365 -in server.csr -CA ca.crt -CAkey ca.key -CAcreateserial -out server.crt`
6. 客户端证书生成流程同上

**证书文件说明：**
| 文件名称 | 用途 |
| :--- | :--- |
| ca.crt | CA根证书，用于验证服务器和客户端证书 |
| ca.key | CA私钥，用于签发证书 |
| server.crt | 服务器证书 |
| server.key | 服务器私钥 |
| client.crt | 客户端证书 |
| client.key | 客户端私钥 |
| crl.pem | 证书吊销列表 |

**客户端设计：**

- 创建SSL上下文，设置验证模式为CERT_REQUIRED
- 连接时验证服务器证书合法性

##### 3.2.4 SSL双向认证模块

**服务端设计：**
- 加载CA证书用于验证客户端证书

**客户端设计：**

##### 3.2.5 安全增强模块

**加密套件支持：**
- ECDHE-RSA-AES256-GCM-SHA384
- ECDHE-RSA-AES128-GCM-SHA256

**数据完整性校验：**

- 发送数据时附带哈希值

### 核心代码说明

#### 4.1 SSL单向认证服务端核心代码

```python
import socket
import ssl
import threading

class SSLServerOneWay:
    def __init__(self, host='0.0.0.0', port=8443):
        self.host = host
        self.port = port
        
    def start(self):
        context = ssl.SSLContext(ssl.PROTOCOL_TLS_SERVER)
        context.load_cert_chain(certfile='../certs/server.crt', 
                               keyfile='../certs/server.key')
        context.set_ciphers('ECDHE-RSA-AES256-GCM-SHA384')
        
        server_socket = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
        server_socket.bind((self.host, self.port))
        server_socket.listen(5)
        
        ssl_socket = context.wrap_socket(server_socket, server_side=True)
        
        while True:
            conn, addr = ssl_socket.accept()
            print(f"SSL协议: {conn.version()}")
            print(f"加密套件: {conn.cipher()}")
            
            thread = threading.Thread(target=self.handle_client, args=(conn, addr))
            thread.start()
```

#### 4.2 SSL双向认证客户端核心代码

```python
import socket
import ssl
import hashlib

class SSLClientMutual:
    def connect(self):
        context = ssl.SSLContext(ssl.PROTOCOL_TLS_CLIENT)
        context.load_cert_chain(certfile='../certs/client.crt', 
                               keyfile='../certs/client.key')
        context.load_verify_locations(cafile='../certs/ca.crt')
        context.verify_mode = ssl.CERT_REQUIRED
        
        sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
        self.ssl_socket = context.wrap_socket(sock, server_hostname=self.host)
        self.ssl_socket.connect((self.host, self.port))
        
    def send_message(self, message):
        data_hash = hashlib.sha256(message.encode()).hexdigest()
        message_with_hash = f"{message}|HASH:{data_hash}"
        self.ssl_socket.send(message_with_hash.encode())
```
